Textspark
=========

Node.js/JavaScript implementation of [@holman](https://github.com/holman)'s
[spark](https://github.com/holman/spark).

Usage
-----

   #!/usr/bin/env node

   var spark = require('spark');

   console.log(spark([1,2,3,4,5,6,7,3]))
